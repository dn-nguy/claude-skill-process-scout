# Process Scout — Skill Template

A starter skill you can customize to become your own weekly automation opportunity finder. It scans the tools you actually use, finds repeatable manual work, scores it by frequency × effort, and helps you turn the highest-value processes into Cowork skills.

## What's in this repo

- **SKILL.md** — the templated skill file with `{{PLACEHOLDERS}}` for your name, tools, and file paths

## How to install and customize it

1. **Download the template.** Grab `SKILL.md` from this repo.

2. **Review it first.** Open the SKILL.md and read through it before installing. Always review community skills before enabling them.

3. **Upload it to Claude.** In Claude, go to **Settings → Capabilities → Skills → Create Skill → Upload**, and upload the SKILL.md file.

4. **Customize it.** In a new Cowork conversation, say:
   > "I just installed the process-scout skill. Help me customize it for my setup."

   Claude will walk through the `{{PLACEHOLDERS}}` in the template and replace them with your actual tools, file paths, and preferences.

5. **Test it.** Say **"what should I automate next?"** or **"run process scout"** to trigger a scan.

6. **Schedule it (optional).** Say **"schedule process scout to run every Monday at 9am"** to get a weekly report automatically.

## What to expect

- Let it run for a week or two before expecting great results — it needs scan history to learn your patterns
- When it surfaces a candidate you accept, it'll draft the new skill for you automatically
- It tracks your accept/reject/defer decisions so it won't resurface ideas you've already passed on
